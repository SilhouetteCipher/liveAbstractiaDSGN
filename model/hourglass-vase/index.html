Finned Hourglass Vases

These sleek hourglass vases feature distinctive vertical fins that create beautiful light and shadow patterns while adding structural interest to any space. The elegant curved silhouette with textured surface makes them perfect for modern and contemporary interiors.

Available in both spiral vase mode for a rapid but decorative-only print, or solid print that's more functional for actual use. The spiral version creates stunning layered effects but I recommend using the adhesive pads linked in the BOM to secure it if you find it too light.

Key Features:

  • Print in your own choice of filament and finish

  • Scale to your preferred size

  • Available in both spiral vase and solid print modes

  • Distinctive finned texture creates visual interest

  • Classic hourglass silhouette suits any decor style

  • Quick print time in spiral vase mode
