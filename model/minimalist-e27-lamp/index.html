Model Details

Modern Cylindrical Table Lamp

Bring a touch of contemporary minimalism to your space with this sleek two-part table lamp design. The clean cylindrical form creates a striking silhouette that works beautifully in modern interiors. This versatile lamp features a simple assembly system - just push the E27 lamp fitting through the base into the top, then secure with the screw fitting.

  • Print in your preferred size and color to match your decor

  • Compatible with standard E27 bulb fittings

  • Easy two-part assembly using the lamp screw fitting

  • Compatible with Bambu lighting kits linked below

  • Minimalist design suitable for any modern space

  • Sturdy base provides excellent stability

This lamp combines function and style with its straightforward cylindrical design. The contrasting base and shade create visual interest while maintaining clean lines. Perfect for bedside tables, living rooms, or office spaces where you want to add a contemporary lighting solution.
