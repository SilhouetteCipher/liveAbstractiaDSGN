Model Details

Dual Tone Pleated Vase

Elevate your space with this modern pleated vase featuring a distinctive dual-tone design. The striking pleated texture creates fascinating light and shadow effects while the two-part construction allows for a perfect color contrast using just a single extruder. This contemporary piece combines clean geometric lines with organic curves for a sophisticated addition to any home decor style.

This vase is available in two printing modes:

  • Spiral vase mode for the top section creates a lightweight decorative piece with rapid print time

  • Solid print mode for the base ensures stability and functionality

    • The clever design features a shroud base that perfectly matches the vase pattern, creating a satisfying interference fit between the two pieces.

    Features:

  • Print in your preferred size and color combination

  • Two-part design creates dual-tone effect with single extruder

  • Choose between spiral vase mode (decorative) or solid print (functional)

  • Perfect interference fit between base and top section

  • Recommended adhesive pads linked in BOM for lightweight spiral versions

  • Clean, modern aesthetic fits various home decor styles

  • Showcase your favorite plants or flowers

I recommend using the adhesive pads linked in the BOM to secure the vase if you find the spiral vase mode version too light.
